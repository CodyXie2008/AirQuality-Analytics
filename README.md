# AirQuality-Analytics

空气质量数据分析项目 - 基于R语言的法国和德国空气质量统计建模

## 项目概述

AirQuality-Analytics 是基于新加坡国立大学(NUS) 2025年冬季学期数据分析和数学统计实践项目(Winter2025-DAMS)的后续研究项目。该项目使用R语言对世界卫生组织(WHO)空气质量数据库中法国和德国多个城市的空气质量指标进行全面的统计分析、可视化和建模。

### 项目演进

- **原始项目**: Winter2025-DAMS - NUS数据分析和数学统计课程实践项目
- **当前项目**: AirQuality-Analytics - 在原始项目基础上的扩展研究和深入分析

AirQuality-Analytics在原有教学项目的基础上，进一步深化了空气质量数据的分析维度，增加了更复杂的统计建模和可视化方法，为环境数据科学研究提供了更全面的分析框架。

## 数据集说明

### 主要数据集：WHO_DB_FR_GER_cleaned_2024.csv

- **数据来源**：世界卫生组织空气质量数据库
- **时间范围**：2010-2021年
- **地理范围**：法国和德国多个城市
- **主要变量**：
  - `pm25_concentration`: PM2.5颗粒物浓度
  - `pm10_concentration`: PM10颗粒物浓度
  - `no2_concentration`: 二氧化氮浓度
  - `country_name`: 国家名称
  - `city`: 城市名称
  - `year`: 年份
  - 人口数据、地理位置等辅助信息

## 分析内容

### 1. 描述性统计分析
- 均值、中位数、方差、标准差计算
- 分位数分析和四分位距(IQR)
- 数据分布特征探索

### 2. 数据可视化
- **条形图**：按年份统计观测数量
- **直方图**：PM2.5浓度分布
- **点图**：PM10浓度在不同年份的分布
- **箱线图**：两国空气质量指标比较
- **散点图**：PM10与PM2.5浓度关系

### 3. 假设检验
- **双样本t检验**：比较德国和法国PM2.5浓度差异
- 检验假设：德国空气质量是否比法国更差

### 4. 线性回归建模
- **简单线性回归**：PM10浓度预测PM2.5浓度
- **多元线性回归**：PM10和NO2浓度共同预测PM2.5浓度
- 模型诊断：残差分析、正态性检验、多重共线性检测

### 5. 预测分析
- 基于回归模型进行空气质量预测
- 不同污染物浓度条件下的PM2.5预测值

## 技术栈

- **编程语言**: R
- **核心包**:
  - `dplyr`: 数据操作和转换
  - `ggplot2`: 数据可视化
  - `car`: 回归分析和方差膨胀因子计算
  - `tidyr`: 数据整理

## 主要发现

### 空气质量趋势
- PM2.5和PM10浓度在2010-2021年间总体呈下降趋势
- 不同城市间的空气质量存在显著差异
- 2020年（疫情期间）空气质量有明显改善

### 国家间比较
- 通过统计检验分析两国空气质量差异
- 可视化展示两国污染物浓度分布特征

### 污染物关系
- PM10和PM2.5浓度存在显著正相关关系
- 建立的回归模型能够有效预测PM2.5浓度
